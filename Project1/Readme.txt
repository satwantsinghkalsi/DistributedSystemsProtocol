ReadMe
======================================
The zip folder contains the following files:
ChandyLamport.java
ConvergeCast.java
Message.java
Node.java
Project1.java
WritingFile.java
launcher.sh
cleanup.sh
config.txt(Sample config file but any valid file can be used)
To run the program place all the contents of the folder in a directory and run the launscher.sh script as:
>>sh launcher.sh <relative path of configuration file> username

The output files are created in the same directory as "config-<node number>.out"
finally run the cleanup.sh to finish
>>sh launcher.sh <relative path of configuration file> username
